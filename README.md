clcov sub-directory contains source code and building instructions of our tool to measure coverage of OpenCL programs.

clmt sub-directory contains source code and building instructions of out tool to generate mutations for OpenCL programs.

Our tools are based on Clang/LLVM infrastructure. Please refer to [http://clang.llvm.org/docs/LibASTMatchersTutorial.html](http://clang.llvm.org/docs/LibASTMatchersTutorial.html) to build dependencies.
